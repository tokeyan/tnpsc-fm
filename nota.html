None Of The Above

If voters in a democratic country do not wish to choose any of the candidates running in the election, voters can choose a button on the voting machine called None of the Above (NOTA - None Of The Above). Rule 49-0 of the Code of Conduct for Elections in India, (1961) describes this process.

ஒரு மக்களாட்சி நடைபெறும் நாட்டில் உள்ள வாக்காளர்கள் தேர்தலில் போட்டியிடும் வேட்பாளர்கள் எவரையும் தேர்வு செய்ய விருப்பம் இல்லை எனில், வாக்காளர்கள் மேற்கண்ட எவரும் இல்லை (NOTA - None Of The Above ) எனும் ஓர் பொத்தானை வாக்கு இயந்திரத்தில் தேர்வு செய்யலாம். இந்திய தேர்தல் நடத்தை விதிகள், (1961)-இல் எனும் சட்டத்தின் விதி எண் 49- 0 இம் முறை பற்றி விவரிக்கிறது.
